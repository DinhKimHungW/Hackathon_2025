### 12.1. Common Issues

#### Issue 1: Cannot Login

**Symptoms:**
- Invalid credentials error
- Page refreshes after login

**Solutions:**
1. Clear browser cache and cookies
2. Verify email is correct (case-sensitive)
3. Reset password via "Forgot Password"
4. Contact admin if account is locked

#### Issue 2: Real-time Updates Not Working

**Symptoms:**
- ğŸ”´ Disconnected status
- Data not refreshing
- WebSocket errors in console

**Solutions:**
1. Check internet connection
2. Refresh the page (F5)
3. Click "Reconnect Now" button
4. Clear browser cache
5. Check firewall/proxy settings

#### Issue 3: Simulation Takes Too Long

**Symptoms:**
- Simulation running > 10 seconds
- Timeout error

**Solutions:**
1. Reduce simulation time range
2. Simplify scenario (fewer changes)
3. Check server load (Admin only)
4. Try again during off-peak hours

#### Issue 4: Cannot Create Schedule

**Symptoms:**
- Validation errors
- "Conflict detected" message

**Solutions:**
1. Check vessel dimensions vs berth capacity
2. Verify ETA is in the future
3. Resolve any detected conflicts first
4. Ensure berth is not in maintenance

#### Issue 5: Reports Not Generating

**Symptoms:**
- "Failed to generate" error
- PDF is blank

**Solutions:**
1. Check date range is valid
2. Ensure data exists for selected period
3. Try different format (Excel instead of PDF)
4. Reduce report complexity
